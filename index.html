Available Commands

!avatar

Display the user's profile picture.

!clear

Clear a specified number of messages in the channel.

!dice

Roll a dice randomly.

!giveaway

Create and manage giveaways in the server.

!leaderboard

Display the leaderboard for the leveling system.

!level

Show the user's current level and points.

!ping

Check the bot's response time.

!poll

Create a poll in the channel.

!quran

Recite verses from the Quran.

!setupchannels

Set up the temporary voice channels system. When using this command, a special voice channel will be created that users can join to create their own temporary voice channels.

!setwelcome

Set the welcome message for new members.

!user

Display information about a specific user.

!vcadd

Add a user to a temporary voice channel.

!vcremove

Remove a user from a temporary voice channel.
